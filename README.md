# TOAST-ImportTool


An import tool to convert XML header at start of motion blocks to texlists for motionrows

The tool can import a TwinCAT3 TOAST project, extract the xml from the motions and inserts them into the MotionRow text list for use in the program
The tool also can export the names of the motion and assemble the logical message for the 5 fault messages for each station
